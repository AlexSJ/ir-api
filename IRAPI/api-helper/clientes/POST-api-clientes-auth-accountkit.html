POST clientes/auth

Breve Descrição

Realiza a autenticação do usuário na API Ingresso Rápido com Account Kit.

Request

POST https://api.ingressorapido.com.br/services/api/v1.2/clientes/auth/accountkit

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8
Json:
  • accessToken: token retornado pelo Account Kit para recuperar os dados do cliente.
Para site e mobile site, apenas
  • guid: identificador único global, gerado randomicamente para evitar ataques CSRF.

Response

Json:
Referência:
  • Retorno: objeto Login com objeto Cliente relacionado
  • AccountKitId: id do usuário do Account Kit
  • Sucesso: booleano indicando o sucesso da operação
  • Mensagem: mensagem de resposta para solicitação de cadastro

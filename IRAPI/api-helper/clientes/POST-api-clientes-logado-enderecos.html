POST clientes/logado/enderecos

Breve Descrição

Método da API que efetua um cadastro de um cliente com base nos dados informados. Retorna um objeto RetornoModel tendo as informações de Sucesso, objeto Login e mensagem informando o erro, quando houver.

Request

POST https://api.ingressorapido.com.br/services/api/v1.2/clientes/logado/enderecos

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8
Json:
Obrigatório:
  • CEP: CEP do usuario, string
  • Endereco: endereço do usuário, string
  • Numero: número do endereço do usuario, string
  • Cidade: cidade do usuario, string
  • Estado: estado do usuario, string
  • EnderecoPrincipal: T: true, ou F: false, marca se endereço principal, char

Response

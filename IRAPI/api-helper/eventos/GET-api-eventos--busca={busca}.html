GET eventos?busca={busca}&localid={localid}&uf={uf}&cidade={cidade}&tipoid={tipoid}&subtipoID={subtipoID}&pg={pg}&qtdpg={qtdpg}&ordem={ordem}

Breve Descrição

https://api.ingressorapido.com.br/services/api/v1.2/eventos?busca=rca&localid=3490&uf=PE&cidade=recife&tipoId=0&subtipoId=62&latitude=-23.5557042&longitude=-46.6627528&distancia=30.509&pg=1&qtdpg=5&ordem=dataAsc

Método da API que retorna a pesquisa informada em {busca}.

  • busca = string da pesquisa (Obrigatório)
  • localid = localidade, integer
  • uf = sigla do estado, string
  • cidade = nome da cidade, string
  • tipoid = identificador do tipo, integer
  • subtipoID = identificador do subtipo, integer
  • latitude = latitude do cliente, double
  • longitude = longitude do cliente, double
  • distância = distância máxima entre o local dos eventos e o locan informado pela latitude e longitude, double
  • pg = número da página da lista paginada, integer
  • qtdpg = número de itens por página, integer
  • ordem = opção de ordenação, sendo dataAsc por data (padrão) e distanciaAsc por distância (a ordenação por distância só funciona se passar também latitude, longitude e distancia)

Request

GET https://api.ingressorapido.com.br/services/api/v1.2/eventos?busca=rca&localid=3490&uf=PE&cidade=recife&tipoId=0&subtipoId=62&latitude=-23.5557042&longitude=-46.6627528&distancia=30.509&pg=1&qtdpg=5&ordem=dataAsc

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8

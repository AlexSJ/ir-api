POST reservas/atual/mesas

Breve Descrição

Reserva uma mesa fechada.

Request

POST https://api.ingressorapido.com.br/services/api/v1.2/reservas/atual/mesas

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8
Json:
Opcional:
  • limparReservaAnterior: limpar reserva anterior, boolean
Obrigatório:
  • apresentacaoID: identificador da apresentação, integer
  • lugarID: identificador do lugar, integer
  • precoID: identificador do preço, integer

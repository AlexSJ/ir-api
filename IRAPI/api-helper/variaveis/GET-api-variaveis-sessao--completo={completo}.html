GET variaveis/sessao?completo={completo}

Breve Descrição

Retorna as variáveis da sessão atual.

  • completo: opcional, traz a compra carregada quando true

Request

GET https://api.ingressorapido.com.br/services/api/v1.2/variaveis/sessao?completo={completo}

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8

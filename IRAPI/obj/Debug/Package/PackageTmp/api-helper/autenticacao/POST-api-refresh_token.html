POST token (refresh)

Breve Descrição

Atualiza o token de autenticação obrigatório para todas as requisições da API.

Request

POST https://api.ingressorapido.com.br/services/api/v1.2/token

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8
Query string:
Obrigatório:
  • refresh_token: token de autorização atual, string
  • grant_type: tipo de permissão que está sendo solicitada pelo aplicativo, string

Response

Json:
Referência:
  • access_token: novo token de autenticação válido,
  • token_type: indica o tipo de token de autenticação
  • expires_in: novo tempo de validade do token
  • userName: usuário de acesso a API
  • .issued: data de emissão do novo token
  • .expires: data de expiração do novo token

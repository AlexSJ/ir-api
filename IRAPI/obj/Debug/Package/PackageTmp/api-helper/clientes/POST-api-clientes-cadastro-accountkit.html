POST clientes/cadastro

Breve Descrição

Realiza o cadastro do usuário na API Ingresso Rápido com Account Kit.

Request

POST https://api.ingressorapido.com.br/services/api/v1.2/clientes/cadastro/accountkit

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8
Json:
Obrigatório
  • Access Token: Token do Account Kit.
  • Nome: Nome do cliente.
  • E-mail: E-mail do cliente.
  • CPF: CPF do cliente.

Response

Json:
Referência:
  • Retorno: objeto Login com objeto Cliente relacionado
  • AccountKitId: id do usuário do Account Kit
  • Sucesso: booleano indicando o sucesso da operação
  • Mensagem: mensagem de resposta para solicitação de cadastro

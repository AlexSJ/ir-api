POST clientes/senha/recuperar

Breve Descrição

Método da API que envia um link com um token para o email do usuário recuperar a senha de acesso. E retorna para o cliente o enderenço email que foi remetido com alguns caracteres escondigos.

Request

POST https://api.ingressorapido.com.br/services/api/v1.2/clientes/senha/recuperar

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8
Json:
Obrigatório:
  • email: email do usuario, string
  • url: url do link, string

Response

Json:

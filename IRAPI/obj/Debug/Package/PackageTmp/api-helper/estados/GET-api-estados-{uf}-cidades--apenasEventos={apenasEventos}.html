Voltar para Índice da API

GET estados/{uf}/cidades?apenasEventos={apenasEventos}

Breve Descrição

Retorna a lista de cidades com pontos de venda filtrado por estado.

  • apenasEventos = true (default) para eventos ativos, false para todos eventos.

Request

GET https://api.ingressorapido.com.br/services/api/v1.2/estados/rs/cidades/

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8

Response

Json:

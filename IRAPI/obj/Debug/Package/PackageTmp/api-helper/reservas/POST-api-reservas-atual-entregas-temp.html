POST reservas/atual/entregas/temp

Breve Descrição

Vincula um endereço temporario de entrega a uma compra.

Request

POST https://api.ingressorapido.com.br/services/api/v1.2/reservas/atual/entregas/temp

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8
Json:
Obrigatório:
  • entregaControleID: identificador do controle de entrega, integer

  • enderecoTemp: apenas deve ser informado quando o entregaControleID passado for de uma entrega do tipo entrega em casa
  • pdvID: apenas deve ser informado quando o entregaControleID passado for de uma entrega do tipo retiradaPDV

Response

Json:

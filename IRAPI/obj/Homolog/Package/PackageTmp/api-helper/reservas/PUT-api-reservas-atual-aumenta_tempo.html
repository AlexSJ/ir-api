Voltar para Índice da API

POST api/reservas/atual/aumenta_tempo/{tempo}

Breve Descrição

Esse método deve atualizar no banco de dados na tabela ingresso da base IngressosNovo e na tabela rq da base SiteIR o tempo de expiração da reserva, adicionando o tempo passado como parâmetro ao tempo que estava la.

Request

PUT https://api.ingressorapido.com.br/services/api/v1.2/reservas/atual/aumenta_tempo

Header:
Obrigatório:
  • ClientInfo: identificador da aplicação, string
  • Authorization: Bearer ...token...
  • Content-Type: application/json
  • Accept-Charset: UTF-8
Json:
Obrigatório:
  • minutos: tempo em minutos para adicionar ao tempo de reserva, integer

Response

Json:

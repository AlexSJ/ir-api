GET tipos/{tipoId}/subtipos?apenasEventos={apenasEventos}

Breve Descrição

Retorna os subtipos de eventos ativos.

  • apenasEventos = true : retorna somente subtipos de eventos disponíveis (Default).
  • apenasEventos = false : retorna todos os subtipos de eventos.

Request

GET https://api.ingressorapido.com.br/services/api/v1.2/tipos/2/subtipos?apenasEventos=true

Header:
Obrigatório:
  • Authorization - token de autorização, string
  • ClientInfo - identificador da aplicação, string
  • Accept-Charset - tipo de codificação, UTF-8

Response

Json:
